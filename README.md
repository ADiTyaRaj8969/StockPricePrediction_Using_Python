# 📈 Stock Price Prediction using LSTM

This project uses a Long Short-Term Memory (LSTM) neural network to predict stock prices based on historical data. It demonstrates time-series forecasting using deep learning techniques in Python.

---

## 🚀 Features

- Fetches real-time stock data using Yahoo Finance API
- Performs data cleaning and preprocessing
- Builds and trains an LSTM model for prediction
- Visualizes actual vs predicted stock trends

---

## 🛠️ Tech Stack

- Python 🐍
- TensorFlow / Keras
- NumPy, Pandas, Matplotlib, Seaborn
- Yahoo Finance (yfinance)
